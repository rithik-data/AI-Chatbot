# 🤖 AI Chatbot Assistant (Python)

An intelligent AI-based chatbot built using Python and Natural Language Processing (NLP).
This chatbot can understand user queries and respond based on predefined intents.

---

## 🚀 Features

* 💬 Interactive command-line chatbot
* 🧠 NLP-based intent recognition using NLTK
* ⚡ Fast and lightweight (pure Python)
* 🔄 Customizable responses using JSON
* 🛠 Easy to extend and improve

---

## 🧠 Technologies Used

* Python
* NLTK (Natural Language Toolkit)
* JSON

---

## 📂 Project Structure

AI-Chatbot/
│── chatbot.py        # Main chatbot logic
│── intents.json      # Training data (intents & responses)

---

## ▶️ How to Run

1. Install dependencies:

```
pip install nltk
```

2. Run the chatbot:

```
python chatbot.py
```

---

## 💡 Example Interaction

```
You: hi
Bot: Hello!

You: what is your name
Bot: I am your AI Assistant

You: help me get job
Bot: Build projects and apply daily
```
